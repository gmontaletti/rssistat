# rssistat 0.1.1

## Installation

You can install the development version of rssistat from GitHub:

```r
# install.packages("devtools")
devtools::install_github("gmontaletti/rssistat")
```

## Usage

```r
library(rssistat)
```

## Development

This package uses renv for dependency management. To set up the development environment:

```r
renv::restore()
```
